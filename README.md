# Homebrew Tap for lintal

A fast Java linter with auto-fix, reading checkstyle.xml configs.

## Installation

```bash
brew tap eleventy7/lintal
brew install lintal
```

## Usage

```bash
# Check a Java project
lintal check /path/to/java/src

# Auto-fix violations
lintal fix /path/to/java/src
```
